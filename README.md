# Sarsan Basketbol

## Project Documentation

This project is designed to facilitate basketball game management and scoring.

### Features
- Player management
- Game scoring and tracking
- Team statistics

## Setup Instructions

To set up the project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/zekeriyakolsuz/sarsan-basketbol.git
   ```

2. Navigate to the project directory:
   ```bash
   cd sarsan-basketbol
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm start
   ```
